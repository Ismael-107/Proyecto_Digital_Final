# Project Analysis - FichajeSecure

## 6a) Strategic objectives
The software aims to improve employee time tracking efficiency and data security. It aligns with digital transformation by replacing manual systems with automated encrypted solutions.

## 6b) Business areas
- HR: manages employee attendance
- Operations: improves workflow tracking
- Communications: ensures transparency

Impact:
Daily operations become faster and more reliable.

## 6c) Digitalization areas
- Time tracking
- Employee management

Improvement:
Reduces human error and increases data accessibility.

## 6d) Integration
Digital HR systems interact with non-digital departments through reports.

Improvement:
Introduce dashboards and reporting tools.

## 6e) Needs
Current:
- Secure tracking
- Easy management

Future:
- Cloud system
- Mobile access

## 6f) Technologies
- Python
- Encryption (cryptography)

Benefits:
- Data protection
- System scalability

## 6g) Security gaps
Possible risks:
- Weak passwords
- Key exposure

Solutions:
- Password hashing
- Secure key storage

## 6h) Data management
Data is stored in encrypted files.

Quality:
- Validation of inputs
- Consistent format
